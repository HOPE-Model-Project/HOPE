## [1.1.0] - 2026-04-08

### Added

- Endogenous representative-day selection (`src/rep_day.jl`): automatically
  select and weight a small set of representative days from a full chronology
  using k-medoids clustering, optional extreme-day augmentation, and
  planning-oriented feature engineering.
- Resource aggregation (`src/aggregation.jl`): group generators by technology
  type and zone to reduce model size while preserving system-level behaviour.
- Holistic GTEP→PCM two-stage planning workflow (`src/run_holistic.jl`):
  `run_hope_holistic` and `run_hope_holistic_fresh` inject GTEP capacity
  decisions into a PCM case and solve both models in sequence.
- EREC (Energy Resilience and Efficiency Criterion) module (`src/erec.jl`):
  compute energy-resilience metrics from PCM or GTEP dispatch output.
- Nodal PTDF-based DC OPF (`network_model = 3`) for PCM.
- Flexible demand response in PCM (`flexible_demand = 1`).
- Shadow-price extraction via optional LP re-solve after MILP solve
  (`write_shadow_prices = 1`).
- `HOPE_MODELCASES_PATH` environment variable to locate model cases stored
  outside the HOPE repository.
- New reference cases: ISO-NE 250-bus nodal PCM (`ISONE_PCM_250bus_case`),
  Germany PCM case suite (nodal and 4-zone), RTS-24 multi-zone congestion
  case, and holistic Maryland full-8760 benchmark.
- Interactive web dashboard (`tools/hope_dashboard`) powered by Plotly Dash.
- GitHub Pages documentation site generated by Documenter.jl.

### Changed

- Default open-source solver switched from Cbc to HiGHS.
- Solver loading is now lazy-guarded; unavailable solvers are skipped
  gracefully without halting package load.
- CI matrix extended to Ubuntu, macOS, and Windows on Julia 1 and nightly.
- `DataFrames` compat broadened from `"1.8"` to `"1"`.
- `DataStructures` compat broadened from `"0.19"` to `"0.18, 0.19"`.

### Fixed

- `using HiGHS` world-age issue resolved by removing `@eval` wrapper.
- CSV `DataFrame` constructor deprecation (positional column argument).
- `NumUnits` column preserved through resource aggregation in PCM.

## [1.0.0] - 2024-02-02

Initial release of the current HOPE Julia package.

### Added

- Generation and Transmission Expansion Planning (GTEP) model.
- Production Cost Model (PCM) with zonal and copper-plate network options.
- Benchmark cases: IEEE-14, IEEE-118, RTS-24, Maryland, and PJM-MD100.
- Multi-solver support: HiGHS, GLPK, Clp, Cbc; optional Gurobi and SCIP.
- XLSX and CSV input data formats.
