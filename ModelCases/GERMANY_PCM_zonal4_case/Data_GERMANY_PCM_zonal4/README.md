# Germany PCM zonal 4-zone case

This Germany zonal case is derived mechanically from the current Germany nodal master case.

- Zones: 50Hertz, Amprion, TenneT, TransnetBW
- Transmission seams: aggregated from nodal cross-zone cutsets
- Generator and storage fleets: aggregated from nodal assets by zone and technology
- Regional load chronology: copied directly from the nodal master regional chronology
- Regional load chronology therefore inherits the promoted base BTM-PV Germany baseline assumptions
- Renewable profiles: copied directly from the nodal master regional wind and solar profiles
- This zonal case is intended as the consistency benchmark against the current nodal Germany baseline
