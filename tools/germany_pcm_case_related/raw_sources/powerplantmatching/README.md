# powerplantmatching Raw Drop

Place the Germany plant extract used for the first-pass fleet build in this folder.

Preferred file names:
- `germany_powerplantmatching.csv`
- `powerplants.csv`
- `powerplantmatching_germany.csv`

Recommended fields if available:
- plant identifier
- name
- country
- fuel / technology
- capacity
- latitude
- longitude
- commissioning / retirement status
